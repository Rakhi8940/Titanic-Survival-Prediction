# Titanic-Survival-Prediction

**Overview**
This project is a binary classification task where the goal is to predict whether a passenger survived or not on the Titanic. The model uses various features like passenger age, class, gender, and fare to make predictions.

**Techniques Used**
Logistic Regression
Decision Trees
Random Forest
Support Vector Machines (SVM)

**Dataset**
The Titanic dataset includes:

Passenger information: age, gender, class, ticket fare, etc.
Survival status: 1 (survived) or 0 (did not survive).
